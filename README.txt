README

Dolphin Emulator and SSX game are needed for this to work

Follow the numbers, starting with 1
	1)  Toshiba Bluetooth driver (for connecting to wiimotes)
			press both 1 and 2 button on wimmote to pair
			press reset button on balanceboard to pair
	
	2)	run as administrator: '2)  disabledriversigning.bat'
	
	3)	install PPJoy
			Once installed, 'configure joysticks' from start menu, then add 'PPJoy Virtual Joystick 2'
	
	4)	place ppjoya.ini and ppjoyb.ini in ~\My Documents\Dolphin Emulator\Config\Profiles
	
	5)	Open Dolphin Emulator
			Configure Gamecube Controller 1 from controllers menu
			load ppjoya.ini
			if that doesnt work*, use ppjoyb.ini

	6) When wiimote and balanceboard are paired via bluetooth, run '4)  glovepie SSX3 load.bat'

	7) either read the header of the script or type '3,b,r' (no commas) before getting on balanceboard

	8) open SSX in Dolphin Emulator
